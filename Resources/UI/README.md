# UI Asset Candidates

These are candidate assets for later UI polish. They are not wired into the app yet.

- `ow-translator-lite-icon.png`: 1024 px O/T app icon with OW-inspired orange, white, charcoal, and cyan accents.
- `ow-translator-lite-icon.ico`: Windows app icon generated from the PNG in common icon sizes.
- `reply-translate.svg`: reply translation action icon concept.
- `ocr-scan.svg`: OCR scan mode icon concept.
- `overlay-window.svg`: overlay window icon concept.

Design direction: dark OW-style utility surface, cyan OCR accents, soft green translation accents, no game logos or copyrighted marks.
